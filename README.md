# KOSPI-Stock-Prediction
[ Visual-Computing ] RNN, LSTM, Transformer를 사용한 주가 예측

<br/>

## Environment
```
Python 3.8
PyTorch 1.8
Cudatoolkit 11.1.
```

## Install
```
pip3 install -r requirements.txt
```

## RNN/LSTM Train & Evaluate & Plot
```
python3 main_rnn_lstm.py
python3 plot.py
```
